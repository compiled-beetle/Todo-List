# Packages Setup

## Project Packages

```bash
npm install --save dotenv @hapi/hapi knex pg joi
```

## Dev Packages

```bash
npm install --save-dev nodemon eslint @hapi/eslint-plugin
```

eslint config

```bash
npm init @eslint/config
```

eslint run on file

```bash
npx eslint file.js

```
